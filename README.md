# 儿童记账小程序 (DaydayUp)

一款专为儿童财商教育设计的记账小程序，帮助家长引导孩子从小养成良好的理财习惯，通过记账、任务奖励等方式，培养孩子的金钱管理意识。

## 功能特性

### 📋 核心功能

1. **多子女管理** - 支持管理多个子女的账户和记账记录
2. **记账功能** - 简单便捷的收入支出记录，包含详细分类
3. **任务奖励** - 通过完成任务获得奖励，寓教于乐
4. **统计分析** - 可视化展示收支情况，帮助孩子了解消费习惯
5. **账户余额** - 实时查看当前账户余额，了解个人财务状况

### 🔐 登录方式

- 微信授权登录 - 一键登录，安全便捷
- 游客模式 - 无需登录即可体验基础功能

### 🎨 设计风格

- 简洁友好的界面设计，适合儿童使用
- 明亮活泼的配色方案，激发学习兴趣
- 直观的图标和引导，降低使用门槛

## 开始使用

### 前提条件

- 微信开发者工具
- 微信小程序账号

### 开发环境设置

1. 克隆或下载本项目到本地
2. 在微信开发者工具中导入项目
3. 填写小程序AppID（或使用测试账号）
4. 编译并运行项目

### 页面导航

- **首页** - 展示当前儿童信息、余额和最近交易记录
- **记账** - 查看所有交易记录和添加新记录
- **统计** - 查看收支统计和分析
- **任务** - 管理任务和查看完成情况

### 主要操作指南

#### 添加儿童
1. 在首页点击儿童名称区域
2. 点击"添加子女"按钮
3. 填写姓名、年龄和性别信息
4. 点击"确定"完成添加

#### 记录收支
1. 在首页点击"记一笔"按钮或记账页面点击"+"按钮
2. 选择交易类型（收入/支出）
3. 输入金额和选择分类
4. 添加描述（可选）和设置日期
5. 点击"保存"完成记录

#### 创建任务
1. 进入任务页面
2. 点击右下角的"+"按钮
3. 填写任务名称、奖励金额和截止日期
4. 添加任务描述（可选）
5. 点击"确定"完成创建

#### 查看统计
1. 进入统计页面
2. 切换不同的时间范围（日/周/月/年）
3. 查看收支概览和分类统计
